Welcome to the User CRUD Application!

This is a simple crud application made with angular, nodeJS/expressJS and postgreSQL.

About

This is an example "SPA" in which you connect to a porgreSQL database to perform the following operations:

  • Create Users and add them to a database
  • Read user data
  • Update user data
  • Delete users from database

Instructions

To use the application, create a database and implement the users table code in the backend/database/database.sql file. Then configure the config object to open a connection pool to your local database, this file can be found in backend/database/db.config.js
